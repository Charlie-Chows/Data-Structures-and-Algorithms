# Stack & Queue

### Stack  (LIFO – Last In First Out) 

| Operation | Meaning            |
| --------- | ------------------ |
| push(x)   | Insert element     |
| pop()     | Remove top element |
| peek()    | See top element    |
| isEmpty() | Check empty        |

### Queue (FIFO – First In First Out)

| Operation  | Meaning        |
| ---------- | -------------- |
| enqueue(x) | Insert element |
| dequeue()  | Remove front   |
| front()    | See front      |
| isEmpty()  | Check empty    |
